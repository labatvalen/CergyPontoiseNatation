Bienvenue dans le README du projet de LABAT Valentin, ZAMORA Mael, GORRICHON Nahia, FROMENT Félix.
Tout d'abord, placez votre dossier PS1 à la racine de votre compte.
Premièrement, connectez vous sur votre terminal à sql.
Deuxièmement, importez le fichiez struce de la bdd en entrant la commande suivante : "source PS1/BDD/structureBDDprojet.sql;"
Ensuite, entrez nimporte qu'elle page du site, vous serz redirigé vers la page de connexion.
Ensuite, entrez dans l'onglet DONNEES et importez le fichier BDD de votre choix.
Enfin, profitez du site.

A noter : tous les fichiers concernant les statistiques sont écrits, mais pas implémentés dans le site car aucune représentation viable n'a été trouvée.
